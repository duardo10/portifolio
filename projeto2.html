Projeto de Construção do Modelo

Este é o "Projeto de Contrução do Modelo Preditivo", onde desenvolvi um Modelo Preditivo em python que seja capaz de identificar se uma imagem é um animal ou um automóvel e predizer qual é o automóvel ou animal. O Modelo Preditivo permite o reconhecimento de animais e automóveis.

O projeto foi desenvolvido utilizando TensorFlow e as tecnologias python de Deep Learning, com integração a um Dataset Público chamado "cifar10" onde encontra-se imagens de automóveis e animais.

Espero que você goste do projeto e fique à vontade para entrar em contato caso queira mais informações!
